Transpose Matrix JavaScript Function

transpose Function:

function transpose(matrix){
    let trans = [];
    for(var i = 0; i < matrix.length; i++){
        let temp = [];
        for(var j = 0; j < matrix[i].length; j++){
            temp.push(matrix[j][i]);                    
        }
        trans.push(temp);
    }
    return trans;
}

Example Usage:

// Example usage of transpose function
console.log(transpose([[1,2,3],[2,3,4],[6,7,8]]));
